# PayNest

PayNest is a modern digital wallet built to make payments simple, secure, and reliable. We’re building PayNest as a safe home for money a place where users can send, receive, and manage funds with confidence.

## 🛠 Tech Stack

- React

- Tailwind CSS

- Lucide react icons

(More integrations coming soon)

## Core Features

- Wallet creation

- Send & receive payments

- Transaction history

- Browser extension support

### Vision

PayNest isn’t just another wallet.

We are building a secure financial layer that can grow into a trusted ecosystem — starting simple, scaling responsibly.
